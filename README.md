# TCP-IP-Socket
winsock을 이용한 TCP/IP 소켓 GUI 응용 프로그램

# 1. 기능

 * 고정+가변길이를 이용한 효율적인 통신
 * 곡선, 삼각형, 사각형, 원 모양 도형 통신
 * Red, Green, Blue 색상 선택
